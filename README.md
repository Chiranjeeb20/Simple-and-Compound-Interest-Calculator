## Formula Used

### Simple Interest

```text
SI = (P × R × T) / 100
```

### Compound Interest

```text
CI = P × (1 + R/100)^T − P
```

Where:

* P = Principal Amount
* R = Rate of Interest
* T = Time (Years)
