# Security Policy

## Supported Versions

This project is currently maintained as a learning and personal portfolio project.

| Version | Supported |
| ------- | --------- |
| 1.0.0   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability or issue related to this project, please open an issue in this repository with the following information:

* Description of the vulnerability
* Steps to reproduce the issue
* Potential impact
* Suggested fix (if available)

## Scope

This project is a simple educational calculator application developed using:

* HTML
* CSS
* JavaScript
* Bash Shell Script

The security policy applies only to the source code contained in this repository.

## Disclaimer

This project is provided for educational and learning purposes. While reasonable efforts are made to maintain code quality, no guarantees are made regarding security, reliability, or fitness for any particular purpose.
